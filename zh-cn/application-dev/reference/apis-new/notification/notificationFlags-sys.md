# notificationFlags    
描述通知标志的实例。  
> **说明**   
>本模块首批接口从API version 8开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本  
    
## NotificationFlagStatus    
    
 **系统能力:**  SystemCapability.Notification.Notification    
 **系统接口：** 此接口为系统接口。    
    
| 名称 | 值 | 说明 |  
| --------| --------| --------|  
| TYPE_NONE | 0 | 默认标志。 |  
| TYPE_OPEN | 1 | 通知标志打开。 |  
| TYPE_CLOSE | 2 | 通知标志关闭。 |
