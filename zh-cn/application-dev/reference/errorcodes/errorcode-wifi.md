# WIFI错误码

## 2401000

**错误信息**

Operation failed.

**错误描述**

WIFI服务内部执行STA相关操作时出现未知错误。

**可能原因**

1. 和WIFI服务建立通信异常。
2. WIFI芯片通信异常。
3. 其他未知错误。

**处理步骤**

1. 重新执行打开或关闭WIFI。
2. 重新执行打开或关闭WIFI，或重启设备。
